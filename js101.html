Основы JavaScript

Учимся программировать прямо в браузере!

HTML CSS JavaScript Главная

1. Подключение JavaScript

Скрипты можно подключить внутри HTML через тег <script>:

<script>
  alert("Привет, мир!");
</script>

2. Переменные (let, const)

Создаём переменные и выводим их в консоль:

let name = "Алекс";
console.log(name);
const year = 2025;
console.log(year);

3. Условия (if)

Выполняем код в зависимости от условия:

let score = 80;
if (score > 60) {
  console.log("Ты прошёл!");
} else {
  console.log("Попробуй ещё раз.");
}

4. Циклы (for)

Выводим числа от 1 до 5:

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

5. Функции

Создаём функцию и вызываем её:

function greet(name) {
  console.log("Привет, " + name + "!");
}
greet("Мир");

6. Работа с DOM

Меняем текст на странице:

// HTML: <p id="demo">Старый текст</p>
document.getElementById("demo").textContent = "Новый текст!";

7. События (onclick)

Реакция на клик по кнопке:

// HTML: <button onclick="sayHi()">Нажми меня</button>
function sayHi() {
  alert("Привет!");
}

8. Массивы

Сохраняем список значений:

let fruits = ["яблоко", "банан", "груша"];
console.log(fruits[0]); // яблоко

9. Объекты

Храним связанные данные:

let user = {
  name: "Игорь",
  age: 30
};
console.log(user.name);

10. Взаимодействие с пользователем

Получаем данные через prompt:

let name = prompt("Как тебя зовут?");
alert("Привет, " + name + "!");

Это лишь основы JavaScript. Дальше — больше: анимации, работа с сервером, игры!
