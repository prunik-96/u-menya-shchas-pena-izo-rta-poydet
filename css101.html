Как писать на CSS

Стили для сайтов — легко и красиво!

HTML CSS JavaScript Главная

CSS описывает внешний вид HTML-элементов. Вот 20 полезных примеров:

1. Цвет текста

p {
  color: red;
}

Делает текст в абзаце красным.

2. Фон

body {
  background-color: black;
}

Задаёт чёрный фон для страницы.

3. Размер шрифта

h1 {
  font-size: 32px;
}

Увеличивает заголовок.

4. Центрирование текста

p {
  text-align: center;
}

Центрирует текст внутри абзаца.

5. Жирный текст

strong {
  font-weight: bold;
}

Делает текст жирным.

6. Рамка

div {
  border: 2px solid white;
}

Добавляет белую рамку вокруг блока.

7. Скругление углов

img {
  border-radius: 10px;
}

Скругляет углы изображения.

8. Тень текста

h1 {
  text-shadow: 2px 2px 5px black;
}

Добавляет тень под текстом.

9. Внутренний отступ

div {
  padding: 20px;
}

Добавляет пространство внутри блока.

10. Внешний отступ

div {
  margin: 20px;
}

Добавляет пространство вокруг блока.

11. Шрифт

body {
  font-family: Arial, sans-serif;
}

Задает шрифт для текста.

12. Псевдокласс :hover

a:hover {
  color: yellow;
}

Изменяет цвет ссылки при наведении.

13. Отображение по центру

div {
  margin: 0 auto;
  width: 50%;
}

Центрирует блок по горизонтали.

14. Плавающие элементы

img {
  float: left;
  margin-right: 10px;
}

Прижимает изображение к левому краю.

15. Флексбокс

.container {
  display: flex;
  justify-content: space-between;
}

Распределяет элементы внутри блока.

16. Скрыть элемент

.hidden {
  display: none;
}

Полностью скрывает элемент со страницы.

17. Позиционирование

.box {
  position: absolute;
  top: 10px;
  left: 20px;
}

Устанавливает точное положение элемента.

18. Анимация

@keyframes fadeIn {
  from {opacity: 0;}
  to {opacity: 1;}
}

.fade {
  animation: fadeIn 2s ease-in;
}

Простая анимация появления.

19. Размеры блока

.box {
  width: 300px;
  height: 200px;
}

Устанавливает размеры для блока.

20. Градиентный фон

body {
  background: linear-gradient(to right, red, orange);
}

Создаёт плавный переход цвета фона.
