Как писать на HTML

HTML (HyperText Markup Language) — язык разметки для создания веб-страниц.

Простой пример:

<!DOCTYPE html>
<html>
  <head>
    <title>Моя первая страница</title>
  </head>
  <body>
    <h1>Привет, мир!</h1>
  </body>
</html>

Совет: Используй редактор кода вроде VS Code и сохраняй файлы с расширением .html.
