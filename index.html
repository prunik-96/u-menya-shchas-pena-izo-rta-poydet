Примеры HTML-кода

20 полезных примеров с пояснениями

HTML Примеры CSS JavaScript Главная

1. Заголовок

Тег <h1> используется для заголовков.

<h1>Привет, мир!</h1>

2. Параграф

Тег <p> создаёт абзац текста.

<p>Это параграф текста.</p>

3. Ссылка

Тег <a> создаёт гиперссылку.

<a href="https://example.com">Перейти</a>

4. Картинка

Тег <img> вставляет изображение.

<img src="image.jpg" alt="Описание">

5. Список (ненумерованный)

<ul> создаёт маркированный список.

<ul>
  <li>Элемент 1</li>
  <li>Элемент 2</li>
</ul>

6. Список (нумерованный)

<ol> создаёт нумерованный список.

<ol>
  <li>Первый</li>
  <li>Второй</li>
</ol>

7. Таблица

Таблица создаётся с помощью <table>.

<table>
  <tr><th>Имя</th><th>Возраст</th></tr>
  <tr><td>Аня</td><td>23</td></tr>
</table>

8. Кнопка

Тег <button> создаёт кнопку.

<button>Нажми меня</button>

9. Форма

Форма для ввода данных.

<form>
  <input type="text" placeholder="Ваше имя">
</form>

10. Комментарий

Комментарии не отображаются на странице.

<!-- Это комментарий -->

11. Линия

<hr> создаёт горизонтальную линию.

<hr>

12. Жирный текст

Для выделения текста жирным.

<b>Важно!</b>

13. Курсив

Для выделения текста курсивом.

<i>Наклонный текст</i>

14. Подчёркнутый текст

<u> подчёркивает текст.

<u>Важная информация</u>

15. Вставка блока

Тег <div> — универсальный контейнер.

<div>Контейнер</div>

16. span

<span> — строчный контейнер.

<p>Текст с <span>выделением</span></p>

17. Тег <br>

Переход на новую строку.

Привет<br>Мир!

18. Встраивание видео

Через тег <iframe>.

<iframe width="560" height="315"
src="https://www.youtube.com/embed/dQw4w9WgXcQ"
frameborder="0" allowfullscreen></iframe>

19. Метаданные

Внутри тега <head>.

<meta charset="UTF-8">

20. Подключение CSS

Тег <link> для подключения стилей.

<link rel="stylesheet" href="style.css">
